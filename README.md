# 🎮 Xbox Game Pass Dashboard

## Sobre o Projeto
Esse projeto consiste na criação de um dashboard interativo no Microsoft Excel para análise de assinaturas do Xbox Game Pass.

O dashboard utiliza:
- Tabelas Dinâmicas
- Gráficos Dinâmicos
- Segmentação de Dados
- KPIs visuais
- Layout personalizado inspirado em dashboards corporativos

---

## 📊 Dados Utilizados
A base de dados contém informações sobre:
- Tipo de assinatura
- Planos do Xbox Game Pass
- Renovação automática
- EA Play Season Pass
- Minecraft Season Pass
- Valores de assinatura
- Cupons aplicados
- Receita total

---

## ⚙️ Funcionalidades do Dashboard
- Filtros interativos
- Análise por tipo de assinatura
- Comparação entre planos
- Indicadores de receita
- Visualização dinâmica de vendas

---

## 🛠 Ferramentas Utilizadas
- Microsoft Excel
- Tabelas Dinâmicas
- Segmentação de Dados
- Fórmulas dinâmicas

---

## ▶️ Como Reproduzir
1. Faça o download do arquivo `.xlsx`
2. Abra no Microsoft Excel
3. Habilite edição, caso solicitado
4. Utilize os filtros laterais para interagir com o dashboard
